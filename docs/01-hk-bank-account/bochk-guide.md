# 中银香港 (BOCHK) 办理指南

> 最后更新时间：2025-11-02  
> 核心摘要：0 门槛、0 管理费，支持 App 线上开户；办理时需"身在香港"。

## 1. 办理前准备
- 本人身处香港（定位校验）。
- 智能手机并安装最新版"中银香港 BoC HK"App。
- 内地身份证、港澳通行证、有效入境记录 PDF（"12367移民局"）。
  - ⚠️ **重要提示**：如提示出入境PDF文件无效，请重新下载最新的入境记录PDF并重新上传。
- 可接收邮寄的收件地址（用于实体卡）。

## 2. 线上开户流程
1) 打开 App → "开立账户"。
2) 证件类型选"内地身份证"。
3) 勾选"我身处香港"→ 选择"即时开立"→ "我不在分行"。
4) 账户类型选"自在理财账户"。
5) 上传材料：入境记录 PDF、身份证正反面、港澳通行证。
6) 人脸识别与信息填写（用途建议"储蓄/投资"；填写收件地址）。
7) 提交审核并设置密码。

## 3. 实体卡收取指南（2025年最新经验）

### 3.1 香港邮政信件收取
⚠️ **重要提醒**：实体卡正常半个月左右到达内地，但**千万不要一直傻等，基本不会主动联系你**。

1. **主动查询**：务必打电话联系 11185，说明有香港平邮一直未收到，留下详细地址。
2. **等待回复**：正常第二天会收到电话回复，当地邮政所会告知取件地点。
3. **快速取件**：直接前往取件地址，寻找带有 HONGKONG POST 贴条的信件即可。实际案例：在村委会的邮政收发室找到的，需要翻查半天信件。

### 3.2 实体卡激活方法
最方便的方法是在内地的任意一台中国银行(BOC)的 ATM 机上操作，全程大约1分钟就能搞定。

1. **插卡**：将中银香港的实体卡插入 ATM 机的插卡口。
2. **输入密码**：输入你开户时设置的6位数字密码（流动保安编码，PIN码）。
3. **随便操作**：在ATM机的界面选择"查询余额"或"账户"等任意一个功能。
4. **取卡**：操作完成后，选择"取卡"即可。系统会自动完成激活，之后卡片就可以正常使用了。

### 3.3 激活注意事项
- **密码别输错**：激活时输入的是6位数字PIN码，不是网上银行密码。连续输错超过3次，卡片可能会被锁定，需要联系银行处理。
- **养卡防休眠**：为防止账户因长期不用而被冻结，建议偶尔存入或取出少量资金，保持账户活跃。

## 4. 其他注意事项
- 办理过程需稳定网络；中断可能重做流程。
- App 多数场景禁止截图，重要信息建议线下记录保管。
- 与内地中行账户互转通常更便捷（以当期政策为准）。
- 开户成功后建议关注中银香港官方通知，及时了解政策变动。
